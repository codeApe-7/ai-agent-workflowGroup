## 评分规则

- **9-10**: 机械化强制执行，无需人工干预
- **7-8**: 有文档约束 + 部分自动化检查
- **5-6**: 仅有文档约束，依赖 Agent 自律
- **3-4**: 约束不完整，偶尔出现漂移
- **1-2**: 无有效约束

## 更新流程

1. 每次 Garbage Collection 扫描后更新评分
2. 每次 Harness 改进后更新对应模块评分
3. 趋势用箭头标记：UP / DOWN / STABLE / NEW
4. 持续低分模块应作为 Harness 改进的优先目标
